# TaskTracker
Приложение предоставляет следующую функциональность:  Реализованы алгоритмы для работы со строками. Приложение может приоритизировать задачи и проверять, не пересекаются ли они по времени выполнения. Создано API приложение и доступ к его методам через HTTP-запросы. Приложение покрыто  Unit-тестами
